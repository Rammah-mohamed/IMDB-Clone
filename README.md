# 🎬 IMDb Clone  

A professional, high-performance IMDb clone built with **React, TypeScript, GraphQL, MongoDB, and Tailwind CSS**. This project allows users to **browse movies, add them to lists, and manage their favorite content** efficiently.  

🚀 **Live Demo:** [IMDb Clone](https://imdb-clone-eta-three.vercel.app/)  
🔧 **Backend API (GraphQL):** [API Server](https://graphql-server-production-1d52.up.railway.app/graphql)  
🔧 **Backend API (MongoDB):** [API Server](https://mongodb-server-production-0356.up.railway.app/)  

---

## ⚡ Features  

✅ **Movie Database Integration** – Fetches movies, actors, and ratings using **GraphQL & TMDB API**  
✅ **User Authentication** – Create an account, log in, and manage watchlists  
✅ **Personalized Movie Lists** – Users can add, update, and remove movies  
✅ **Advanced Search** – Search for movies, TV shows, and actors  
✅ **Optimized Performance** – 90+ Lighthouse score with image compression via **Sharp**  
✅ **Responsive UI** – Built with **Tailwind CSS** for seamless mobile & desktop experience  
✅ **Vitest Testing** – Unit and integration tests with **high test coverage**  

---

## 🛠️ Tech Stack  

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Apollo Client  
- **Backend:** GraphQL (Apollo Server), Express, MongoDB  
- **Image Optimization:** Sharp (served via an Express-based API)  
- **State Management:** Context API  
- **Testing:** Vitest & React Testing Library
